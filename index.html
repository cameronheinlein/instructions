How to Create a Personal Webpage Using Github and VS Code

Step 1

  1. Navigate to Github.com
  2. Choose Sign Up to create a free profile or Sign In with your existing account
  3. Choose New
  4. Under Create a New Repository, give your Repository a name. This is how you will naviagte to the page later. You have the option to add a description too.

    5. You might get this page. Just select Upload an existing file to upload from VS Code editor or creating a new file to create your code right in Github.

  5. Drag and drop your files from your Finder, or select choose your files to browse and choose from a folder.
  6. Select Commit Changes
